# نظام الإدارة المالية

موقع ويب بسيط وفعال لإدارة الأعضاء والعمليات المالية والتقارير.

## الميزات

- ✅ **إدارة الأعضاء**: إضافة وحذف الأعضاء مع تفاصيلهم
- ✅ **إدارة الفرق**: إنشاء وإدارة الفرق المختلفة
- ✅ **إدارة الصناديق**: تتبع الصناديق والأهداف المالية
- ✅ **الإيدات والسحوبات**: تسجيل جميع العمليات المالية
- ✅ **التقارير**: عرض تقارير شاملة عن الإيدات والسحوبات والأرصدة
- ✅ **التخزين المحلي**: جميع البيانات تُحفظ محلياً على جهازك
- ✅ **بدون خادم**: يعمل مباشرة من المتصفح بدون الحاجة لخادم ويب

## البنية

```
simple-web/
├── index.html           # الصفحة الرئيسية

├── reports.html         # صفحة التقارير
├── style.css            # التصميم المشترك
```

## كيفية الاستخدام

### الطريقة 1: فتح مباشر
1. انسخ مجلد `simple-web` إلى جهازك
2. افتح ملف `index.html` بنقرة مزدوجة أو من المتصفح
3. ابدأ الاستخدام مباشرة

### الطريقة 2: من خادم محلي (اختياري)
إذا واجهت مشاكل في فتح الملف مباشرة، يمكنك استخدام خادم محلي:

```bash
# باستخدام Python 3
python -m http.server 8000

# أو باستخدام Python 2
python -m SimpleHTTPServer 8000

# ثم افتح المتصفح على: http://localhost:8000
```

## الصفحات

### 1. الصفحة الرئيسية (index.html)
- ملخص سريع للإحصائيات
- أيقونات للتنقل إلى الصفحات الأخرى
- عرض التاريخ والوقت الحالي

### 2. الأعضاء (members.html)
- **الأعضاء**: إضافة وحذف الأعضاء
- **الفريق**: إدارة الفرق المختلفة
- **الصندوق**: إدارة الصناديق والأهداف المالية

### 3. الإيداع (deposit.html)
- نموذج لتسجيل إيداع جديد
- قائمة بآخر الإيدات

### 4. السحب (withdraw.html)
- نموذج لتسجيل سحب جديد
- قائمة بآخر السحوبات

### 5. التقارير (reports.html)
- **تقرير الإيدات**: عرض جميع الإيدات
- **تقرير السحوبات**: عرض جميع السحوبات
- **تقرير الأرصدة**: عرض أرصدة جميع الأعضاء

## التخزين المحلي

جميع البيانات تُحفظ محلياً في متصفحك باستخدام `localStorage`:
- الأعضاء
- الإيدات
- السحوبات
- الفرق
- الصناديق

**ملاحظة**: البيانات تبقى محفوظة حتى لو أغلقت المتصفح أو أعدت تشغيل الجهاز.

## حذف البيانات

لحذف جميع البيانات المحفوظة:
1. افتح أدوات المطور (F12)
2. اذهب إلى Console
3. اكتب: `localStorage.clear()`
4. اضغط Enter

## التصميم

- **الألوان**: درجات رمادية ناعمة وجميلة
- **الخط**: تجوال (Tajawal)
- **حجم الخط**: صغير وسهل القراءة
- **حجم الحاوية**: 90% من عرض الصفحة
- **الاستجابة**: يعمل على جميع أحجام الشاشات
